## 1. 产品概述

独居老人远程关怀面板是一款面向家属和社区照护人员的实时监控与关怀平台，帮助远程掌握独居老人的健康状况、用药情况和安全状态，及时发现异常并做出响应。

- 解决独居老人无人实时照护的安全隐患，降低意外风险
- 目标用户：老人家属、社区照护人员、居家养老服务方

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家属 | 邀请注册 | 查看老人数据、接收告警、管理用药提醒、联系其他家属 |
| 照护人员 | 管理员分配 | 查看老人数据、处理告警、更新健康记录 |

### 2.2 功能模块

1. **首页概览**：老人状态总览、今日健康摘要、待办事项、快捷导航
2. **健康记录**：血压/心率/血糖/体温趋势图表、历史记录查询
3. **用药提醒**：今日用药计划、用药状态标记、提醒时间设置
4. **异常告警**：实时告警通知、告警分级、历史告警记录
5. **家属联系人**：联系人列表、紧急呼叫、关系标注

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页概览 | 状态卡片 | 展示老人当前状态（正常/注意/告警），心跳、血压、体温等关键指标速览 |
| 首页概览 | 今日待办 | 显示今日待服药物、待处理告警等待办事项列表 |
| 首页概览 | 快捷入口 | 提供各功能模块的快捷跳转卡片 |
| 健康记录 | 趋势图表 | 展示血压、心率、血糖、体温等指标的趋势折线图 |
| 健康记录 | 历史列表 | 按日期查看历史健康数据记录，支持筛选 |
| 用药提醒 | 今日用药计划 | 按时间线展示今日需服药物，标记已服/未服/漏服状态 |
| 用药提醒 | 用药总览 | 展示所有药物的服用计划、剂量、频次 |
| 异常告警 | 实时告警 | 展示当前未处理的异常告警，按严重程度分级（紧急/警告/提示） |
| 异常告警 | 历史告警 | 展示已处理的告警历史记录，支持按时间和级别筛选 |
| 家属联系人 | 联系人列表 | 展示所有家属联系人的姓名、关系、电话，支持一键拨打 |
| 家属联系人 | 紧急联系 | 置顶显示紧急联系人，醒目标识 |

## 3. 核心流程

用户打开面板后，首先进入首页概览，快速了解老人当前状态。若存在异常告警，顶部以醒目颜色提示，用户可点击进入告警详情。日常流程为查看健康数据趋势 → 确认今日用药情况 → 处理告警 → 联系家属。

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：暖橙 + 柔和绿（传达温暖关怀与安全健康）
- 辅助色：珊瑚红（告警）、天蓝（信息提示）
- 按钮风格：圆角柔和，带微阴影，hover 有温和放大效果
- 字体：标题使用 Noto Sans SC 粗体，正文使用 Noto Sans SC 常规
- 布局风格：左侧导航栏 + 右侧内容区，卡片式布局
- 图标风格：线条图标，统一使用 lucide-react

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页概览 | 状态卡片 | 大圆角卡片、柔和渐变背景、关键指标数字加粗突出、微动画呼吸效果 |
| 首页概览 | 今日待办 | 列表卡片、左侧彩色边条标识优先级、复选框交互 |
| 首页概览 | 快捷入口 | 图标+文字网格卡片、hover 阴影加深 |
| 健康记录 | 趋势图表 | 自定义折线图、渐变填充区域、时间轴选择器 |
| 健康记录 | 历史列表 | 表格布局、斑马纹行、筛选标签 |
| 用药提醒 | 今日用药计划 | 垂直时间线、圆形状态图标（绿/灰/红）、药物名称+剂量 |
| 用药提醒 | 用药总览 | 药物卡片网格、每张含药物名/剂量/频次/下次服用时间 |
| 异常告警 | 实时告警 | 左侧彩色竖条（红/橙/黄）、告警时间+描述+操作按钮 |
| 异常告警 | 历史告警 | 筛选栏+列表、时间倒序排列 |
| 家属联系人 | 联系人列表 | 头像+姓名+关系标签+电话、紧急联系人置顶高亮 |
| 家属联系人 | 紧急联系 | 红色边框卡片、电话图标按钮、一键拨打 |

### 4.3 响应式设计

- 桌面优先设计，大屏利用多列布局展示更多信息
- 平板适配：导航栏可收起为图标模式，内容区自适应
- 移动端：底部 Tab 导航替代侧边栏，卡片单列排列
